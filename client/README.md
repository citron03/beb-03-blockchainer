# Blockchainer 클라이언트 앱 🐣
- Create React App

## 실행 방법 🐤
> npm install

- node modules을 설치합니다.

> npm run start

- client app을 실행합니다.

## 사용한 라이브러리 🐥
* react ^17.0.2
* styled-components ^5.3.5
* react-router-dom ^5.3.0
* axios ^0.26.1
* redux ^4.2.0
* react-redux ^8.0.1
* @reduxjs/toolkit ^1.8.1
